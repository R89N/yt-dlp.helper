
[Features]
- A GUI is provided to simplify downloading using YT-DLP

v0.1 : First version

[Installation]
- Download YT-DLP : https://github.com/yt-dlp/yt-dlp
- Extract(Install) YT-DLP
- Download YT-DLP.HELPER and extract to the same folder with YT-DLP

[Usage]
- Run YT-DLP.HELPER
- Copy video address to text input area
- Click download!

[Options]
- if YT-DLP option file exists, options will apply to downloads
